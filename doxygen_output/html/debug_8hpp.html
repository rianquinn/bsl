Debug

Uses fmt to output a debug statement. Debug statements are synchronized with other debug statements using a global mutex and debug statements can be filtered using the level template parameter. If the provided level is <= to the global debug level, and NDEBUG is not defined, the debug statement is outputted.

Fatal

Uses fmt to output a fatal statement. Fatal statements are synchronized with other debug statements using a global mutex. Fatal statements do not have a debug level, and how a fatal statement is handled depends on whether or not AUTOSAR compliance is enabled. Note that unlike the other debug statements, you must provide a source location, and this debug statement will insert a newline for you.
