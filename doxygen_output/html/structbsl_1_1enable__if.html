Detailed Description

template<bool B, typename T = bsl::int_fast32_t>
class bsl::enable_if< B, T >

Defines a conditional "if" statement for SFINAE overload resolution. If B is true, bsl::enable_if has a public member typedef of type T, otherwise, there is no public member typedef.

Note
Unlike the std::enable_if, the default type is an integer, not a void. This allows for SFINAE to set default values to 0, and not * = nullptr, which is far more verbose, while still supporting the * = nullptr syntax for legacy code.
