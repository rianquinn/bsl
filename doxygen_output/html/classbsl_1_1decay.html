Detailed Description

template<typename T>
class bsl::decay< T >

Applies lvalue-to-rvalue, array-to-pointer, and function-to-pointer implicit conversions to the type T, removes const-qualifiers, and defines the resulting type as the member typedef type. Note that this does not remove volatile as the BSL does not support the use of volatile.

namespace bsl
{

example_decay_overview() noexcept
{

}
}
}
#endif
