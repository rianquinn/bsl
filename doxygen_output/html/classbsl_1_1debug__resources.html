Detailed Description

This class provides the global resource used by the debug statements. For the debug statements to work, this class must be created so that the debug statements have something to output to. If this class is not created, debug statements will be ignored.

print

This is the internal implementation of the print function that all of the other debug functions use to output debug statements. All debug statements are synchronized using a global mutex, and (other than fatal) all debug statement provide the ability to filter which debug statements are outputted using a global debug level. An optional thread ID can be added to the output by defining BSL_THREAD_ID as "foo();".

print exception

This function prints an exception that might come from the print function to stderr using fputs. Note that error codes from fputs are ignored so if this function fails to execute, you will not receive an error message.
